# Organizm
Browser game project

### Installation
```bash
git clone https://github.com/umpire-3/Organizm.git
cd Organizm
npm install
```

### Running
```bash
node main.js
```

### Controls
- Move mouse while holding left button to look around 
- `W` `A` `S` `D` - movement 
- `SPACE` - hold position
